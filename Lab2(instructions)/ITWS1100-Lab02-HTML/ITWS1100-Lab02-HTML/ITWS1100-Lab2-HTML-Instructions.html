Create a resume

  • I've provided you a starting HTML template and an example resume.
  • Setup and Instructions:
    • You should create a folder on your machine for this class. (ie ITWS1100)
    • Then create a folder within that folder for the programming portion of the class (ie iit)
    • Then create a folder within that folder for your lab(s) (ie lab2)
    • Then save the HTML template to your machine, placing it in that folder for the lab
    • Now edit it to create a resume like the example. You should enter your own information in the resume. Your information doesn't have to be perfect, but make your best effort; you can update and correct the dates and titles on your own.
    • After you submit your lab, you will be taking this resume down to the career center for review - so make it good!
  • Guidelines:
    • Name your file: yourRCSID-FullName-resume.html
    • Keep your code in one document. Your CSS should be embedded on the page (in the <head> section)
    • Don't use tables: you don't need them. Hint: imagine blocking the content with boxes. Some of your boxes will need to float, and some will need to be given explicit widths.
    • Use proper indentation (see HTML/CSS technology lecture notes)
    • Your markup must be valid
    • Use the web, e.g. W3Schools (html, css), for reference.
    • You will be graded based on the the Lab 2 rubric.
  • We will work through some of this today in class
  • Feel free to experiment and be creative - but show me that you can create something similar to the example resume - this needs to be a reume template for you to use in the job seeking process.
  • Submission:
    • This lab is due by end of date specified in LMS
    • Make sure to create a readme file with any comments/observations about the lab
    • Zip up your lab2 folder and submit to LMS prior to EoD on date due
    • Make sure you take your resume to the career center for review
